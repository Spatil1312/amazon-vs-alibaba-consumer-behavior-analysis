# Global E-Commerce Consumer Behavior Analysis: Amazon vs. Alibaba

## 📌 Project Overview
This project presents a comparative data-driven analysis of consumer behavior on two global e-commerce giants: **Amazon** (representing Western e-commerce frameworks) and **Alibaba** (representing Eastern e-commerce frameworks). 

The core of this research focuses on user experience design, purchasing patterns, and the emerging impact of **Generative AI shopping assistants** on reducing consumer decision fatigue and cart abandonment rates.
